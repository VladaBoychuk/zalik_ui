# Додаток погоди (Weather App)

Мобільний додаток для перегляду погоди в реальному часі, розроблений на React Native з використанням Expo. Інформація про погоду та координати міст завантажується через OpenWeatherMap API.

## Основні можливості

* **Двомовний пошук міст:** можна шукати міста як українською, так і англійською мовою.
* **Динамічний фон:** колір градієнта на головному екрані автоматично змінюється залежно від поточної погоди (ясно, хмарно, дощ, сніг, туман).
* **Детальна інформація:** відображає температуру, відчуття (feels like), вологість, швидкість вітру, тиск та видимість.
* **Автодоповнення:** при введенні назви міста у пошуковому рядку відображаються підказки з назвами регіонів та країн.

## Вимоги

Для запуску додатку локально вам знадобляться:
* Node.js (рекомендовано версію LTS)
* Встановлений додаток Expo Go на вашому смартфоні (для тестування на iOS чи Android)
* Безкоштовний API-ключ від [OpenWeatherMap](https://openweathermap.org/)

## Інструкція із запуску

1. **Встановіть залежності:**
   ```bash
   npm install
   ```

2. **Налаштуйте змінні оточення:**
   Створіть файл `.env` у кореневій папці проєкту та додайте туди ваш ключ OpenWeatherMap:
   ```env
   EXPO_PUBLIC_OPENWEATHER_API_KEY=ваш_api_ключ_тут
   ```

3. **Запустіть сервер розробки:**
   ```bash
   npx expo start
   ```

4. **Відкрийте додаток:**
   * Відскануйте QR-код на екрані за допомогою камери смартфона (iOS) або додатку Expo Go (Android).
   * Або натисніть клавішу `w` у терміналі, щоб запустити додаток у веб-браузері.

## Структура проєкту

```
├── App.tsx                          # Головний компонент додатку
├── src/
│   ├── components/
│   │   ├── SearchBar.tsx            # Поле пошуку міста
│   │   ├── WeatherCard.tsx          # Картка з температурою та іконкою
│   │   ├── WeatherDetails.tsx       # Деталі погоди
│   │   └── LoadingSpinner.tsx       # Індикатор завантаження
│   ├── services/
│   │   └── weatherApi.ts            # Робота з API
│   ├── types/
│   │   └── weather.ts               # TypeScript інтерфейси
│   └── constants/
│       ├── theme.ts                 # Кольори, шрифти та стилі
│       └── i18n.ts                  # Тексти та локалізація
├── app.json                         # Конфігурація Expo
├── package.json                     # Залежності проєкту
└── .env                             # API-ключ (не додається в репозиторій)
```
